目标： 拒绝“肉眼看图”，用概率论来判断一个策略是否真的挣钱。

推荐书籍：

《Introductory Econometrics for Finance》- Chris Brooks: 专门为金融设计的计量经济学，比纯数学书好读得多。重点看时间序列分析（ARIMA/GARCH）。

《利用 Python 进行数据分析》(Python for Data Analysis) - Wes McKinney: 主要是精读 Pandas 处理时间序列（Resampling, Timezone, Window functions）的部分。

维基百科 (Wikipedia) 必看词条：

Mean reversion (均值回归)

Standard deviation (标准差/波动率)

Sharpe ratio (夏普比率)

Hypothesis testing (假设检验：判断策略是否靠运气)
