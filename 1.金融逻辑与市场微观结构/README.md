目标： 理解钱是怎么流动的，以及交易背后的“对手盘”是谁。

推荐书籍：

《投资学》(Investments) - Zvi Bodie: 经济学小白的“圣经”。重点看资产配置、资产定价模型（CAPM）和有效市场假说。

《交易与交易所》(Trading and Exchanges) - Larry Harris: 深入理解订单簿、流动性和买卖价差。这能帮你理解为什么你的回测结果在实盘中可能买不到。

维基百科 (Wikipedia) 必看词条：

Financial market (金融市场概念)

Order book (订单簿机制)

Arbitrage (套利逻辑)

Modern portfolio theory (现代组合理论)
