目标： 将你的 10 年 Python 功力转化为生产力，构建可回测、可执行的系统。

推荐书籍：

《Quantitative Trading》- Ernest P. Chan 绰号“陈老师”的入门神作。作者是物理学家转量化，逻辑极其清晰，非常适合程序员。

《Python 股票量化交易》(作者多为国内作者，如丁鹏或徐毅) 建议找一本针对 A 股环境的书，了解 T+1 制度、涨跌停限制等中国特色逻辑。

维基百科 (Wikipedia) 必看词条：

Backtesting (回测及其陷阱)

Technical analysis (技术指标的数学本质)

Pairs trading (配对交易)
